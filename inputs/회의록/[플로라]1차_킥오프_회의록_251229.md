## 2. 요구사항 논의

### 2.1 핵심 기능 (Must-have)

#### 상품 관리
- 일반 상품: 생화, 드라이플라워, 화분, 가드닝 용품
- **정기구독 상품**: 월간 플라워 박스 (새로운 기능)
  - 구독 주기: 2주/4주 선택
  - 구독 회차별 가격 할인 (1회차 정가, 3회차 이상 10% 할인)
- 상품 옵션: 사이즈(S/M/L), 포장 옵션(기본/선물)

#### 회원 시스템
- 소셜 로그인: **카카오, 네이버** (필수)
- 회원 등급: 일반 / 그린멤버 / VIP
  - 그린멤버: 누적 10만원 이상, 적립 2%
  - VIP: 누적 30만원 이상, 적립 3%, 무료배송
- 적립금 시스템: 구매 시 적립, 리뷰 작성 시 추가 적립

#### 결제/배송
- **토스페이먼츠** 연동 (카드, 계좌이체, 간편결제)
- 배송비: 3만원 이상 무료, 미만 시 3,000원
- **희망 배송일 선택** 기능 (생화 특성상 필수)
- 제주/도서산간 추가 배송비 5,000원

#### 구독 결제
- 정기결제 시스템 필요
- 구독 일시정지/해지 기능
- 다음 결제일 알림 (카카오 알림톡)

### 2.2 추가 기능 (Nice-to-have)

- 선물하기 기능 (받는 분 정보 별도 입력)
- 꽃말 검색 기능
- 오늘의 추천 (AI 기반?)
- 인스타그램 피드 연동

### 2.3 관리자 기능

- 상품 등록/수정 (Shopify Admin 기본)
- 주문/배송 관리
- **구독 현황 대시보드**
- 회원 등급 관리
