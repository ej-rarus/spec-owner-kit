# ChiHoo Terrarium 종합예약정보 대시보드

> 작성일: 2024-12-24

---

## 1. 프로젝트 개요

### 1.1 서비스명
**ChiHoo Terrarium 종합예약정보 대시보드**

### 1.2 서비스 유형
웹 서비스 (반응형 웹 애플리케이션)

### 1.3 목적
여러 채널(네이버 예약, 솜씨당, 수기입력 등)에서 들어오는 **예약 정보를 한 곳에 통합**하여 관리

### 1.4 사용자
- 내부 팀 사용 (관리자 대시보드)
- 여러 직원이 함께 사용

## 2. 핵심 기능

### 2.1 예약 목록/캘린더 뷰
| 기능 | 설명 | 우선순위 |
|------|------|----------|
| 예약 목록 보기 | 전체 예약을 리스트로 조회 | P1 |
| 캘린더 뷰 | 월간/주간/일간 캘린더로 예약 현황 확인 | P1 |
| 필터/검색 | 날짜, 상태, 예약 채널별 필터링 | P1 |
| 예약 상태 표시 | 예정/완료/취소 등 상태 구분 | P1 |

### 2.2 예약 등록/수정
| 기능 | 설명 | 우선순위 |
|------|------|----------|
| 수기 예약 등록 | 전화/문자 예약 직접 입력 | P1 |
| 예약 정보 수정 | 날짜, 시간, 인원 변경 | P1 |
| 예약 취소 | 취소 처리 및 사유 기록 | P1 |
| 예약 채널 표시 | 네이버/솜씨당/수기 등 출처 구분 | P1 |

|------|------|----------|
| 고객 목록 | 고객 정보 조회 | P2 |

## 5. 기술 스택

### 5.1 프론트엔드
| 기술 | 선택 | 비고 |
|------|------|------|
| 프레임워크 | Next.js 14+ (App Router) | React 기반 |
| 스타일링 | Tailwind CSS | 빠른 UI 개발 |
| UI 컴포넌트 | shadcn/ui | 커스터마이징 용이 |
| 상태관리 | Zustand 또는 React Query | 필요시 |
| 캘린더 | react-big-calendar 또는 FullCalendar | 검토 필요 |
