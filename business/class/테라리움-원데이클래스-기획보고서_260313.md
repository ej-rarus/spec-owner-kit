## 1. 시장 트렌드 분석

### 1.1 글로벌 테라리움 시장 규모

| 연도 | 시장 규모 | 비고 |
|------|----------|------|
| 2024년 | **13.4억 달러 (약 1.8조 원)** | 기준 연도 |
| 2033년 | **29.2억 달러 (약 3.9조 원)** | 전망치 |
| 연평균 성장률 (CAGR) | **8.9%** (2025-2033) | |
| 아시아태평양 CAGR | **10.2%** | 가장 빠른 성장 지역 |

### 1.2 한국 원데이클래스 시장

| 지표 | 수치 |
|------|------|
| 공예 분야 시장 규모 | 약 **10조 원** |
| 서울 지역 원데이클래스 시장 | 약 **1.3조 원** |
| 월평균 매출 성장률 (플랫폼 기준) | **45%** |
| 홈가드닝 매출 성장 (SSG닷컴) | 전년 대비 **97% 급증** |

### 1.3 인기 테라리움 유형

| 유형 | 특징 | 인기도 |
|------|------|--------|
| **이끼 테라리움 (밀폐형)** | 유리병 안에 이끼·양치류·소형 열대식물. 자체 수분순환으로 관리 용이 | 가장 높음 (원데이클래스 1순위) |
| **다육 테라리움 (개방형)** | 다육식물/선인장 중심, 건조 환경, 공기 순환 필요 | 높음 (초보자 선호) |
| **밀폐형 미니어처 테라리움** | 피규어/미니어처 장식 + 이끼로 스토리텔링 | 높음 (SNS 바이럴) |
| **팔루다리움** | 육지+물 결합, 습지 생태계 재현 | 중간 (심화 수요) |
| **비바리움** | 소동물(양서류 등) 사육 환경 구현 | 니치 마켓 |

> 원데이클래스에서는 **이끼 테라리움(밀폐형)**이 압도적 1위. 관리가 쉽고 완성도가 높으며 2시간 내 완성 가능.

### 1.4 SNS 인기 스타일

- **미니어처 스토리텔링**: 사람/동물 피규어를 배치하여 작은 세계관을 만드는 스타일
- **감성 무드등 테라리움**: LED 조명 내장형, 인테리어 소품 겸용
- **시즌별 테마**: 크리스마스 테라리움, 벚꽃 테라리움 등 시즌 한정
- **캐릭터 콜라보**: 지브리, 디즈니 등 캐릭터 피규어 활용
- **투명 유리구 테라리움**: 행잉형, 구형 유리 용기 활용

### 1.5 시장 성장 동인

1. **도시화/1인가구 증가** → 소규모 실내 자연 수요
2. **디지털 피로** → 물성 있는 체험 선호 (트렌드 코리아 2025 "물성 매력")
3. **반려식물 문화 확산** → 테라리움으로 자연스러운 진입
4. **기업 팀빌딩 체험화** → 회식 문화에서 체험 문화로 전환, B2B 수요 확대
5. **웰니스/힐링 트렌드** → 원예치료(Horticultural Therapy) 수요 증가

## 2. 경쟁 현황

### 2.1 주요 경쟁 공방

| 공방 | 위치 | 특징 |
|------|------|------|
| 해피에버애프터 | 서울 인사동 | 네이버 예약 연동, 관광지 입지 활용 |
| 이끼다움 | 서울 건대 | 단체 출강 전문, 인스타그램 마케팅 활발 |
| 아이엠모스(Iammoss) | 서울 마곡 | 이끼 테라리움 전문, 인스타 기반 |
| 푸르다 | 서울 | 가드닝 클래스 + 화분 쇼핑몰 겸업 |
| 싱그루 | 온라인+오프라인 | 밀폐형 이끼 테라리움 전문, 대량 주문 할인(100개+) |
| 어포레스트 | 온라인 | 테라리움/비바리움 키트 및 용품 전문 쇼핑몰 |

### 2.2 플랫폼별 현황

- **솜씨당**: 200만 유저, 전국 2만개+ 클래스 (테라리움 다수 등록)
- **모카클래스**: B2B 단체 클래스 전문, 기업 워크숍 연계
- **움클래스**: 체험권 기반 플랫폼

### 2.3 시장 가격대

| 구분 | 가격 범위 | 비고 |
|------|-----------|------|
| 개인 클래스 (1-5인) | 40,000~70,000원 | 기본 이끼 테라리움 기준 |
| 소규모 단체 (6-7인) | 50,000원/인 | 단체 할인 시작 |
| 중규모 단체 (8-10인) | 30,000원/인 | |
| 대규모 단체 (15인+) | 25,000원/인 | 기업 워크숍 최적 |
| 프리미엄/대형 | 80,000~120,000원 | 대형 용기, 고급 재료 |
| DIY 키트 (온라인) | 9,800~48,700원 | 배송비 별도 |

### 2.4 경쟁사 차별화 성공 포인트

- **출강 서비스**: 기업/학교/기관으로 직접 방문하여 진행
- **플랫폼 연동**: 네이버 예약, 솜씨당, 모카클래스 등 다중 채널 노출
- **관광지 입지**: 인사동, 한남동, 연남동 등 유동인구 많은 지역
- **키트+클래스 하이브리드**: 오프라인 체험 후 온라인 키트 판매로 연결
- **시즌 한정 클래스**: 크리스마스, 발렌타인, 어버이날 등 테마 클래스

## 3. 소비자 트렌드

### 3.1 MZ세대 취미 클래스 소비 패턴

| 지표 | 수치 |
|------|------|
| 원데이클래스 참여 경험 (20-59세) | **76.9%** |
| 2회 이상 참여율 | **82.7%** |
| 수강 의향 | **82.3%** |
| 나만의 취미활동 욕구 | **88.8%** |
| 원데이 vs 정기 클래스 비중 | 원데이 **약 80%** |

### 3.2 타겟 고객층 분석

| 타겟 | 비율(추정) | 특징 |
|------|-----------|------|
| **핵심: 20-30대 여성** | 40-45% | 커플 데이트, 친구 모임, SNS 콘텐츠 |
| 개인 (힐링 목적) | 15-20% | 자기계발, 스트레스 해소 |
| 기업 단체 (10인+) | 20-25% | 팀빌딩, 워크숍, 반나절 선호 |
| 가족 | 10-15% | 주말/방학, 아이 동반 |
| 40-60대 여성 | 확장 타겟 | 원예치료 효과, 몰입 경험 |

### 3.3 계절별 수요 패턴

| 계절 | 수요 수준 | 특징 |
|------|----------|------|
| **봄 (3-5월)** | 최고 성수기 (150%) | 식물 관심 증가, 커플·기업 시즌 |
| **여름 (6-8월)** | 비수기 (60-70%) | 실내 활동, 방학 가족 체험 |
| **가을 (9-11월)** | 성수기 (120-130%) | 데이트 시즌, 기업 하반기 워크숍 |
| **겨울 (12월)** | 최성수기 (160%) | 크리스마스/연말 선물, 송년 단체 |
| **겨울 (1-2월)** | 비수기 (50-60%) | 새해 특강, 온라인 집중 |

### 3.4 재방문 및 온라인 연계

- 테라리움은 타 공예(캔들, 도자기) 대비 **재방문 잠재력이 높은 편** (다양한 용기/식물 조합, 난이도 확장)
- **DIY 키트 판매**: 오프라인 체험 후 온라인 키트 구매로 자연 연결
- **관리법 콘텐츠**: 인스타그램/유튜브를 통한 테라리움 관리 팁
- **플랜테리어 연계**: 홈가드닝 매출 급증 추세와 맞물림

### 3.5 원예치료 과학적 근거

| 연구 | 결과 |
|------|------|
| 서울대-국립정신건강센터 | 원예활동 주 2회 이상 시 스트레스 인지율 **28% 감소**, 우울감 완화 |
| 텍사스 A&M 대학 | 식물 돌봄이 코르티솔(스트레스 호르몬) 저하 효과 |
| 일본 치바대학 | 녹색 식물 관찰만으로 뇌파 안정, 불안감 완화 |

### 4.2 프로그램 1 — "나의 첫 이끼정원" (입문)

| 항목 | 내용 |
|------|------|
| **타겟** | 초보자, 식물 키우기 경험 없는 분, 1인 가구 |
| **시간** | 90분 |
| **난이도** | ★ |
| **결과물** | 유리돔 테라리움 (지름 12cm, 높이 15cm) |
| **인원** | 2~8명 |
| **추천 시즌** | 연중 상시 (특히 봄·가을) |
| **가격대** | 38,000~45,000원 |

**포함 재료**: 유리돔 용기(12cm), 화산석·자갈(배수층), 마사토(색상 3종 택1), 야자 활성탄, 배양토, 비단이끼 1종, 소형 관엽식물 1포트, 미니어처 피규어 1개, 꾸미기 재료(색모래·조약돌), 분무기, 관리법 안내 카드

**커리큘럼**:

| 단계 | 시간 | 내용 |
|------|------|------|
| 1. 오리엔테이션 | 10분 | 테라리움 원리(물 순환, 광합성), 이끼 생태 설명, 완성 예시 감상 |
| 2. 재료 소개 | 10분 | 각 재료의 역할 설명, 색모래·피규어 선택 |
| 3. 배수층 만들기 | 15분 | 자갈 > 활성탄 > 마사토 레이어링, 경사 디자인 팁 |
| 4. 식재 | 20분 | 배양토 위 관엽식물 배치, 비단이끼로 지면 덮기, 핀셋 사용법 |
| 5. 데코레이션 | 15분 | 미니어처, 색모래, 조약돌로 개성 있는 연출 |
| 6. 마무리 | 10분 | 적정 수분량 시연, 빛/온도/분무 주기 안내 |
| 7. 포토타임 | 10분 | 포토존 촬영, 안전 포장 |

### 4.3 프로그램 2 — "이끼 마운틴 스케이프" (중급)

| 항목 | 내용 |
|------|------|
| **타겟** | 테라리움 경험자, 디자인 관심, 인테리어 취미 |
| **시간** | 120분 |
| **난이도** | ★★★ |
| **결과물** | 대형 유리 자(jar) 테라리움 (지름 18cm, 높이 22cm) — 산악 풍경 연출 |
| **인원** | 2~6명 |
| **추천 시즌** | 연중 상시 (특히 여름 — 초록이 가장 생기 있는 시기) |
| **가격대** | 58,000~68,000원 |

**포함 재료**: 대형 유리 자(jar) 용기(18cm), 배수층 재료(화산석·활성탄·마사토), 배양토, 이끼 3종(비단이끼·서리이끼·꼬리이끼), 소형 관엽식물 2포트, 자연석(산 형태 연출용), 유목(드리프트우드) 소형, 미니 피규어 2~3개, 전문 도구 세트(긴 핀셋·스포이트·흙밀대), 관리법 안내 카드

**커리큘럼**:

| 단계 | 시간 | 내용 |
|------|------|------|
| 1. 이끼 심화 학습 | 15분 | 이끼 3종 생태적 특성, 질감·색감 비교, 실물 관찰 |
| 2. 디자인 구상 | 10분 | 산악 풍경 레퍼런스 감상, 스케치 |
| 3. 지형 만들기 | 20분 | 배수층 + 배양토로 산·계곡 지형 설계 |
| 4. 식재 & 레이아웃 | 25분 | 자연석 배치, 식물 식재, 3종 이끼 질감별 배치로 원근감 |
| 5. 유목 & 디테일 | 15분 | 드리프트우드로 고목 연출, 피규어 스토리 부여 |
| 6. 수분 조절 | 15분 | 밀폐형/반밀폐형 차이, 최적 습도 세팅 |
| 7. 완성 감상 | 10분 | 작품별 품평, 장기 관리법(곰팡이 대처, 가지치기) |
| 8. 포토타임 | 10분 | 촬영 및 패키징 |

| 항목 | 내용 |

### 7.5 만족도 운영 팁

1. **"완성 보장" 원칙** — 시간 내 못 끝내는 수강생에게 추가 10~15분 여유. 절대 재촉하지 않기
2. **실패 없는 설계** — 어떻게 만들어도 예쁘게 나오도록 재료 조합을 사전 검증
3. **과정 사진 촬영** — 강사가 수강생의 "작업하는 손" 사진을 촬영하여 클래스 후 공유
4. **이름 부르기** — 예약 시 수집한 이름으로 호칭. 소규모 클래스의 장점 극대화
5. **5분 이끼 스토리** — 클래스 중간에 이끼 관련 흥미로운 사실 공유
6. **포장의 품격** — 브랜드 박스+완충재+관리법 카드+스티커
7. **애프터케어 메시지** — 1주 후 "이끼 잘 자라고 있나요?" 카카오톡 → 재방문 전환
