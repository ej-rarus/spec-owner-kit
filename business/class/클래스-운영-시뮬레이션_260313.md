## 4. 재료 자유도 — 기본 고정 + 프리미엄 업그레이드 모델

### 기존안 vs 확정안

| | 기존안 (선택형) | **확정안 (업그레이드형)** |
|---|---------------|----------------------|
| 구조 | 이끼·식물 3종 중 택1 | 기본 세트 고정 + 프리미엄 추가금 |
| 이끼 | 3종 중 선택 | 비단이끼 기본 포함 / **프리미엄 이끼 추가 시 +α** |
| 식물 | 3종 중 선택 | 기본 식물(피토니아 등) 포함 / **프리미엄 식물 추가 시 +α** |
| 피규어 | 10~15종 자유 선택 | **제공 안 함** — 가져오고 싶은 분만 터치 |
| 원가 관리 | 3종씩 재고 | **1종 기본 + 프리미엄 소량 재고** |

### 확정: 기본 세트 구성

**기본 세트 (수강료에 포함)**:
- 유리 용기 (소형 or 중형)
- 비단이끼 (기본 이끼)
- 기본 식물 1포트 (피토니아 등 대표가 선정)
- 배수층 재료 (화산석 + 활성탄 + 배양토)
- 색모래·장식 자갈
- 포장재
- 차 (음료 제공)

**프리미엄 업그레이드 옵션 (추가금)**:
- 프리미엄 이끼 추가: 서리이끼, 꼬리이끼 등 → **+3,000~5,000원**
- 프리미엄 식물 추가: 희귀종, 대형 식물 → **+3,000~8,000원**

**피규어**: 공방에서 제공하지 않음. 수강생이 직접 가져오면 배치 가능.

> **해피에버애프터 체험 결과 피드백**: 피규어를 주력으로 하면 수강생을 만족시킬 수 없었음. 이끼·식물 자체의 완성도에 집중하는 것이 맞다.

### 재고 관리 기준

| 재료 | 준비 기준 | 재고 부담 |
|------|----------|----------|
| 비단이끼 (기본) | 주 단위 입고, 밀폐 보관 | 낮음 (2~3주 유지) |
| 기본 식물 1종 | 주 1회 입고, 주간 예상 수강생 수만큼 | 낮음 (1종만 관리) |
| 프리미엄 이끼 | 소량 보관, 예약 주문 시만 준비 | 매우 낮음 |
| 프리미엄 식물 | 소량 보관, 예약 주문 시만 준비 | 매우 낮음 |
| 배수층·색모래 | 월 1회 대량 | 없음 (반영구) |

> **핵심**: 기본 1종씩만 관리하면 되므로 로스율과 재고 부담이 최소화. 프리미엄은 예약 기반으로 운영.

| | 계산 |

## 8. 월 매출 1,000만원 달성 방법

### 방법 1: 클래스만으로 (중형 60,000원 기준)

- 1,000만 ÷ 60,000 = **167명/월**
- 주 6일, 4주 = 24일
- 167명 ÷ 24일 = **일 평균 7명** (클래스 1.5회, 평균 4.6명)
- **→ 평일 1회(4~5명) + 주말 2회(5~6명)로 충분히 달성 가능**

### 방법 2: 클래스 + 프리미엄 믹스

| 구성 | 횟수 | 인원 | 단가 | 매출 |
|------|------|------|------|------|
| 중형 클래스 | 28회 | 5명 | 60,000원 | 8,400,000원 |
| 초대형 클래스 | 2회 | 3명 | 140,000원 | 840,000원 |
