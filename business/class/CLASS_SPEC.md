# 치후 테라리움 클래스 기획서

> 최종 업데이트: 2026-02-04

---

## 1. 개요

### 1.1 목적
- 다양한 고객층에게 테라리움 제작 경험 제공
- 힐링과 창작의 즐거움을 통한 브랜드 가치 전달
- 원데이클래스부터 정기 코스까지 폭넓은 프로그램 운영

### 1.2 브랜드 컨셉
- **"작은 숲을 품다"** - 일상 속 자연의 위안
- 빈티지 보타니컬 감성
- 정성과 여유가 느껴지는 경험

### 1.3 운영 정보
- **주소**: 서울특별시 성북구 삼양로 38 101호
- **운영시간**: 화-일 10:00-20:00 (월요일 휴무)
- **최대 정원**: 기본 6명 (프로그램별 상이)
- **예약 채널**: 사이트, 네이버예약, 솜씨당, DM, 전화
