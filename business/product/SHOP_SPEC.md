# 치후 테라리움 상품 기획서

> 최종 업데이트: 2026-02-06

---

## 1. 개요

### 1.1 목적
- 클래스 외 상시 매출 채널 확보
- "알고 사는 이끼" — 도감(Atlas)과 연동된 차별화 커머스
- 클래스 수강 후 → 집에서도 계속하는 경험 연결
- 선물/기프트 시장 공략

### 1.2 상품 철학
- **"숲을 배달합니다"** — 작은 자연을 일상으로
- 단순 재료 판매가 아닌, 경험과 이야기가 담긴 상품
- 치후 브랜드 감성 (빈티지 보타니컬, 따뜻한 자연)
- 받는 사람이 감동하는 패키징

### 1.3 판매 채널
- **Phase 1**: 사이트 카탈로그 + 카카오 채널 주문
- **Phase 2**: 사이트 내 직접 결제 (PG 연동)
- **병행**: 네이버 스마트스토어, 카카오 선물하기 입점 검토

### 1.4 배송 정보
- 택배 발송 (CJ대한통운/우체국택배)
- 살아있는 식물: 화~금 발송 (주말 물류센터 체류 방지)
- 배송비: 3,500원 / 50,000원 이상 무료배송
- 제주/도서산간: +3,000원

```

└── 관리법 카드 + 도감 QR 카드
```

### 17.2 원가 절감 포인트
- 유리 용기: 대량 발주 시 개당 30~50% 절감
- 이끼: 자체 재배 시 원가 거의 제로
- 패키징: 크라프트 박스 + 스티커 조합 (소량 시작 가능)
- 가이드북: 초기 소량 디지털 인쇄 → 주문량 증가 시 옵셋 인쇄 전환
